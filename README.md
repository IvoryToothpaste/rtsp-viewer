# rtsp-viewer
Simple FFmpeg Tkinter-based RTSP viewer for IP cameras. Features a scrollable stream sidebar, hotkey support, active feed highlighting, and threaded OpenCV video playback.<br/>
Configurable by config.json, url, stream name, hotkeys, and comment. <br/>
<br/>
FULL TRANSPARENCY: This was AI-aided by Google Gemini.</br>
<br/>
Also in the config.json are a bunch of actual rtsp streams from South Korean traffic cameras https://www.data.go.kr/data/15063717/fileData.do which is pretty cool.
